# core-python-assignment
# Python Mini Systems Collection

This repository contains **basic Python programs that simulate real-world systems** using functions and simple data structures.
These examples are useful for learning **problem-solving, functions, lists, dictionaries, and basic program design**.

---

## Projects Included

### 1. E-Commerce Cart System

A program that calculates the total price of items in a shopping cart.

**Features**

* Calculates total price using a function
* Applies **10% discount if more than 5 items**
* Handles empty cart cases

**Concepts Used**

* Dictionaries
* Functions
* Conditional statements

---

### 2. Restaurant Menu Management

A program that manages a restaurant menu.

**Features**

* Add new menu items
* Remove existing items
* Check item availability
* Handles invalid removal attempts

**Concepts Used**

* Lists
* Functions
* Conditional checks

---

### 3. Classroom Performance Tracker

Tracks student marks and identifies the top performer.

**Features**

* Calculates **average marks**
* Determines **top-performing student**

**Concepts Used**

* Dictionaries
* Functions
* Loops
* Built-in Python functions

---

### 4. Movie Ticket Booking System

Simulates seat booking in a cinema hall.

**Features**

* Book seats
* Cancel seat bookings
* Prevent duplicate booking
* Display available seats

**Concepts Used**

* Lists
* Functions
* List comprehension

---

### 5. Hospital Patient Management

Stores and searches patient records.

**Features**

* Stores patient information

**Concepts Used**

* List of dictionaries

---

### 6. Customer Feedback Analysis

Analyzes customer ratings.

**Features**

### 7. Taxi Fare Calculation

Calculates fares for multiple taxi trips.

**Features**

**Concepts Used**

* Functions

## How to Run

1. Clone the repository

```
git clone https://github.com/your-username/python-mini-systems.git
```

2. Navigate to the project folder

```
cd python-mini-systems
```

3. Run any Python file

```
python filename.py
```
